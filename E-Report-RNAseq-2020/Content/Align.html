Alignment Tool: HISAT2
HISAT2 is a fast and sensitive alignment program for mapping next-generation sequencing reads to genomes.
The new indexing scheme of HISAT2 is based on hierarchical graph FM index (GFM index). HISAT2 using global GFM index and a large set of small GFM indexes that collectively cover the whole genome for rapid and accurate alignment.
Furthermore, small GFM index sets come with a great performance for splicing junction alignment, so HISAT2 could be an effective tool for transcriptome alignment. To learn more about GFM indexes, refer here for details.
