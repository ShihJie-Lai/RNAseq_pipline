By default, we set p-value cut off to 0.05 and abs(log2FC) cut off to 1 for discovering differential genes.
P-value calculated by DEseq with non-grouped sample used "blind mode",
which all samples were treated as replicates of a single global "condition" and used to build one model for statistic test.
The q-value of a test measures the proportion of false positives incurred (called the false discovery rate) when that particular test is called significant.
Without enough replicates for a test, q-value might over-estimate the FDR, so we would not set a cutoff of q-value in our data by defalut.

Analysis Process
  • Differential expression analysis was performed using StringTie (StringTir v2.1.4) and DEseq (DEseq v1.39.0) or DEseq2 (DEseq2 v1.28.1) with genome bias detection/correction and Welgene Biotech's in-house pipeline.
  • Functional enrichment assay in differentially expressed genes of each experiment design was performed using clusterProfiler v3.6.
  • Genes with low expression level (< 0.3 TPM value) in either or both of the treated and control samples were excluded.
  • Genes with p value ≤ 0.05 and ≥ 2-fold changes were considered significantly differentially expressed.
