GO: BP (Biological Process)

Bar Plot
It depicts the enrichment scores (e.g. p values) and gene count or ratio as bar height and color.

pyramid Plot
lx and rx are the values specifying the left and right extents of the left and right bars respectively.

Dot plot
Dot plot is similar to bar plot with the capability to encode another score as dot size.

Gene-Concept Network
In order to consider the potentially biological complexities in which a gene may belong to multiple annotation categories and provide information of numeric changes if available

Enrichment Map
Enrichment map organizes enriched terms into a network with edges connecting overlapping gene sets.

goplot
Goplot can accept output of GO analysis and visualized the GO analysis induced graph.

GO Hierarchy Plot
GO Hierarchy plot by significant enrichment term. Significant level descend from "red" to "yellow".
